# 🕵️ Lie Detector Game

A fun JavaScript-based Lie Detector Game that analyzes a user's typing behavior and generates a random "truth probability" score. This project is created for entertainment purposes only and does not actually detect lies.

## 🚀 Live Demo

**Live Link:** https://ratnasingh90999-prog.github.io/lie-detector-project/

---

## 📌 Features

* Random truth-related questions
* Typing behavior analysis
* Backspace tracking
* Typing speed calculation
* Truth probability score generation
* Suspicious / Probably Truth verdict
* Local Storage support for saving last score
* Responsive and simple UI

---

## ⚙️ How It Works

1. A random question is displayed.
2. The user types an answer.
3. The application records:

   * Typing speed
   * Number of backspaces used
   * Time taken to answer
4. A score is generated based on these factors and a small random factor.
5. The result displays:

   * Truth Probability (%)
   * Verdict
